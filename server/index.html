  3. Scroll down the share menu and tap Add to Home Screen.
  4. The name will show "Calculator". Tap Add.
  5. Done! The app is now on your home screen. You can close Safari.

Instructions for other devices

iPhone / iPad

Open in Safari (not Chrome). Tap the Share button ⬆︎ at the bottom, then Add to Home Screen.

Android

Open in Chrome. Tap the ⋮ menu, then Add to Home screen or Install app.

Windows / Mac / Linux

Open in Chrome or Edge. Click the install icon in the address bar, or go to Menu → Install.

Firefox

Firefox does not support installing web apps. Use Chrome, Edge, or Safari instead.
