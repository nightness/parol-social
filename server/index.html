  3. Scroll down and tap Add to Home Screen.
  4. Tap Add in the top right corner.
  5. The app now appears on your home screen as Calculator.

Instructions for other devices
  1. iPhone/iPad: Open in Safari → tap Share ⬆︎ → Add to Home Screen.
  2. Android: Open in Chrome → tap ⋮ menu → Install app / Add to Home Screen.
  3. Windows/Mac/Linux: Open in Chrome or Edge → click install icon in address bar.
